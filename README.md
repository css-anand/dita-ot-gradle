DITA-OT Gradle Plugin
=====================

A Gradle plugin for publishing DITA documents with [DITA Open Toolkit][dita-ot].

See [https://eerohele.github.io/dita-ot-gradle](https://eerohele.github.io/dita-ot-gradle).
